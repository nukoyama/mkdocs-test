# General info

## MkDocs

- template language: Jinja
- base: Python Markdown

- key words: mkdocs, material, mkdocs+, pymdown, python-markdown, jinja




## Theme

結構テーマ依存性がある模様。

ここにある説明は `material` というテーマにん限った話が殆どです。

```yaml
theme:
  name: material
  language: ja
  features:
    tabs
  palette:
    primary: teal
    accent: teal
```
